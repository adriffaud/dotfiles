## Setup

- Application Launcher: `rofi`
- Bar: `polybar`
- Compositor: `compton`
- Icons: `Paper`
- Music Player: `Spotify`
- Notifications: `dunst`
- Shell: `zsh`
- Terminal Emulator: `tilix`
- Text Editor: `vscode`|`vim`
- Theme: `Arc Darker`
- Web Browser: `Google Chrome`|`Mozilla Firefox`
- Window Manager: `i3`
- Linux Distribution: `Debian Linux`
